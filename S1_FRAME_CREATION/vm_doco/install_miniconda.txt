#### Create Miniconda environment to run S1 burst scripts (for running on Raijin only)

## uses *.yml file which contains list of required packages. The package versions work with the python scripts


1. Download installer (linux 64-bit)
     - https://docs.conda.io/en/latest/miniconda.html

2. Install Miniconda
     > cd /g/data1/dg9/SENTINEL-1_BURSTS
     > bash Miniconda3-latest-Linux-x86_64.sh
         - install to: /g/data1/dg9/SENTINEL-1_BURSTS/miniconda3
         - don't update .bashrc

3. Create new environment with miniconda_environment.yml file
     > source /g/data1/dg9/SENTINEL-1_BURSTS/miniconda3/etc/profile.d/conda.sh
     > conda update conda
     > conda create -n S1_Frames python=3.7.3
     > conda activate S1_Frames
     > conda env update --name S1_Frames --file miniconda_environment.yml

# load environment manually
    > source /g/data1/dg9/SENTINEL-1_BURSTS/miniconda3/etc/profile.d/conda.sh
    > conda activate S1_Frames

# deactivate
    > conda deactivate
